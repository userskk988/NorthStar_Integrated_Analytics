# NorthStar Urban Mobility & Logistics - Coursework Submission

## 1. Project Overview
This submission provides a complete analytical solution for NorthStar, addressing fragmented operational data through both Relational (SQL/R) and NoSQL (MongoDB) architectures.

## 2. Directory Structure
- **`NorthStar_Master_Submission.ipynb`**: Contains implementation for Task 3 (Processing), Task 4 (NoSQL), and Task 5 (Optimization).
- **`task1_2_analytics.R`**: Contains implementation for Task 1 (SQL in R) and Task 2 (R Stats).
- **`mongodb_remodel.json`**: The new NoSQL data structure representing integrated customer case histories.
- **`final_analysis.md`**: Business analysis and findings based on the data.
- **`task5_optimization_report.md`**: Technical report on indexing strategies and performance tuning.
- **`cleaned_*.csv`**: Refined, consistent datasets ready for immediate analysis.

## 3. Implementation Instructions
1.  **Python:** Upload `.ipynb` and `.csv` files to Google Colab.
2.  **R:** Create an R runtime in Google Colab and paste the code from `task1_2_analytics.R`.
3.  **GitHub:** Push this entire directory to a structured repository.
